# Sistema de Login com Recuperação de Senha

## 🚀 Tecnologias
- React
- Node.js
- Express
- MySQL
- Nodemailer

## 🔐 Funcionalidades
- Cadastro de usuário
- Login com sessão
- Logout
- Recuperação de senha por email
- Reset de senha com token

## ▶️ Como rodar

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
